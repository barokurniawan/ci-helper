CONTRIBUTING
==================

## For Member

- Clone your repository
    ```
    git clone https://github.com/USERNAME/ci-helper
    ```
- Commit changing
- Push
- Pull Request

## For Non Member

- Fork this repository
- Clone your repository
    ```
    git clone https://github.com/USERNAME/ci-helper
    ```
- Commit changing
- Push
- Pull Request

## Open Issues

- Please open the issue if you find damage to the source code
